Функция отладки, которая печатает входную переменную, если флаг равен true. Состояние флага устанавливается в main.cpp но видно во всём проекте.

Эта шаблонная функция принимает логический флаг и переменную любого типа. Он печатает значение переменной использование Serial.println(), если флаг установлен. Если флаг ложный, функция ничего не делает.

Пример использования:

int number = 42;
bool debug_flag = true;
print_debug(debug_flag, number);
